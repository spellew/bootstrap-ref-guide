Alert Code 


<div class="alert alert-success" role="alert"> Alerts are just divs with their classes set to "alert", text is placed within these divs. There are three types of alerts other than "alert-sucess", for example "alert-info".
    You can use <strong>alert-sucess</strong>to show a sucessful action done by the user.
</div>

You can use alert-sucess to show a sucessful action done by the user.
You'll definitely want to use alert-warning to signal something that can potentially go bad.
You can use alert-danger to signal something hazardous to the user.

Alert Links


<a href="#" class="alert-link">This is an "alert-link".</a>
When wanting to use a link within an alert, give the a tag the "alert-link" class, so you'll get something like this.

Dismissible Alerts


<button type="button" class="close" data-dismiss="alert">
Alerts can be dismissed by adding a button with "close" as a class, and with data-dismiss set to "alert".

Default Label 

<span  style="font-size: 5vh; color: black; background-color: white;" class = "label label-default">Default Label</span>

To change a label's size, give the <span> element a "font-size" attribute in CSS. To change the color of the text, change the "color" attribute in CSS. To change actual color of the label, change the "background-attribute" in CSS.

Glyphicons




    <span class="glyphicon glyphicon-apple"></span>  
    This creates an apple icon. An apple isn't the only glyphicon though, there are over 250. All you have to do is create a span element, give it "glyphicon" as a class, and give it another class depending on the icon you want. Bootstrap glyphicons should only be used between <span> tags, and the span should have no text or child elements. A complete list of glyphicons can be found here.





    <span style="font-size: 50px;" class="glyphicon glyphicon-apple"></span>  
    To change the size of a glyphicon, give the <span> element a "font-size" attribute in CSS. In this case, we gave it a height of 50 pixels.





    <span style="color: black;" class="glyphicon glyphicon-apple"></span>  
    You can change the color of a glyphicon too! All you have to do is give the <span> element a "color" attribute in CSS.

<script type="text/javascript">
function reset_div() {
    $("#transitions_top").removeClass(); The $() function can also be written as jQuery(), it searches the HTML document for elements that match the parameter. The parameter in this case is an id, it can also be a class. When jQuery finds this parameter it will run the removeClass() function on the found object, and ALL classes will be removed, because the function reveived NO parameters. 
}
function change_width() {
    $("#transitions_top").addClass("width_changed"); Instead of removeClass(), we'll use addClass() to add a class to the element that matches the parameter of the $() function. When the class is added, the transition propert will do all the work.
}
function change_border() {
    $("#transitions_top").addClass("border_changed");
}
function change_color() {
    $("#transitions_top").addClass("color_changed");
}
function change_opacity() {
    $("#transitions_top").addClass("opacity_changed");
}
function change_shadow() {
    $("#transitions_top").addClass("shadow_changed");
}
</script>
<style type="text/css">
#transitions_top {
    width: 34vh; When "margin" is set to "auto", a div is horizontally centered. vh is a unit that web developers use, 1vh is equal to 1% of the available screen space.
    height: 34vh;
    margin: auto;
    background-color: red; 
    border: 8px solid black;
    opacity: 100;
    margin-top: 8vh;
    transition: width 1.4s, border 1.4s, background-color 1.4s, opacity 1.4s, box-shadow 1.4s; Transition is the css property that makes everything happen, you give it another css property and the time it should take to change the property. When the property is changed, it smoothly transitions into the new from the old.
}
.width_changed {
    width: 68vh !important; These are the changes that we'll make to the box. We'll apply these classes to the box using jquery, and the transition property will do all the work. !important is used to make the browser override a previous CSS rule. So, the browser will override the property that existed before we applied the class, and smoothly transition into an updated property.
}
.border_changed {
    border: 16px solid white !important;
}
.color_changed {
    background-color: blue !important;
}
.opacity_changed {
    opacity: 0.25 !important;
}
.shadow_changed {
    box-shadow: 20px 20px 5px green !important;
}
</style>
